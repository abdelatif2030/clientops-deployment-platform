# ClientOps Deployment Platform

Production-style DevOps automation platform using Terraform, Ansible, Docker, Nginx, Prometheus, Grafana, and AWS EC2.

## Stack
- AWS EC2
- Terraform
- Ansible
- Docker
- Nginx
- Flask
- Prometheus
- Grafana
- Node Exporter
- Jenkins

## Project Goal
Automate infrastructure provisioning, server configuration, application deployment, and observability setup.

## Architecture
- App Server
- Monitoring Server
